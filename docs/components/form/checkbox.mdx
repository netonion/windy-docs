# Checkbox

> Checkboxes allow the user to select one or more items from a set.

```jsx title="Import"
import { Checkbox } from "@webeetle/windy";
```

## Default

```jsx live
<div className="flex space-x-5">
  <Checkbox name="Checkbox1" label="With Label" />
  <Checkbox name="Checkbox1" />
  <Checkbox disabled name="Checkbox1" label="Disabled" />
</div>
```

## Colors

```jsx live
</div>
```
