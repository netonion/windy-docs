# Radio

> Radio buttons allow the user to select one option from a set.

```jsx title="Import"
import { Radio } from "@webeetle/windy";
```

## Default

```jsx live
<div className="flex space-x-5">
  <Radio name="radio1" label="With Label" />
  <Radio name="radio1" />
  <Radio disabled name="radio1" label="Disabled" />
</div>
```

## Colors

```jsx live
<div className="md:flex space-x-0 md:space-x-5">
  <Radio color="gray" name="radio2" label="gray" />
  <Radio color="red" name="radio2" label="red" />
  <Radio color="yellow" name="radio2" label="yellow" />
  <Radio color="green" name="radio2" label="green" />

  <Radio color="purple" name="radio2" label="purple" />

</div>
```
