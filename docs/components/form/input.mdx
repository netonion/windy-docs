# Input

> Input fields let users enter and edit text.

```jsx title="Import"
import { Input } from "@webeetle/windy";
```

## Default

```jsx live
<div className="flex space-x-5 justify-center">
  <Input type="text" placeholder="input md" />
  <Input type="text" disabled placeholder="disabled" />
</div>
```

## Colors

```jsx live
<div className="flex space-x-5 justify-center">
  <Input type="text" color="red" placeholder="input md" />
  <Input type="text" color="red" disabled placeholder="disabled" />
</div>
```

## Size

```jsx live
<div className="flex-col w-1/4 space-y-5 justify-center">
  <Input type="text" size="xs" placeholder="input xs" />
  <Input type="text" size="sm" placeholder="input sm" />
  <Input type="text" size="md" placeholder="input md" />
  <Input type="text" size="lg" placeholder="input lg" />
  <Input type="text" size="xl" placeholder="input xl" />
</div>
```

## Prefix/Suffix

```jsx live
<div className="md:flex justify-center space-x-0 md:space-x-5 space-y-5 md:space-y-0 items-center">
  <div>
    <Input fullWidth prefix="prefix.com/" type="text" placeholder="Prefix" />
  </div>
  <div>
    <Input fullWidth type="text" suffix="suffix.com" placeholder="Suffix" />
  </div>
</div>
```
