# Avatar

> Generate an avatar for show the status of ypur users

```jsx title="Import"
import { Avatar } from "@webeetle/windy";
```

## Default

```jsx live
<div className="flex w-full space-x-5 justify-center items-center">
  <Avatar src="https://images.unsplash.com/photo-1535713875002-d1d0cf377fde?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=400&q=80" />
</div>
```

## Text

```jsx live
<div className="flex w-full space-x-5 justify-center items-center">
  <Avatar layout="text" text="RT" />
</div>
```
