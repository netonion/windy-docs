# Media

> Generate a Media component useful for cards, panels and chat

```jsx title="Import"
import { Media } from "@webeetle/windy";
```

## Default

```jsx live
<div className="flex w-full space-x-5 justify-center items-center">
  <Media
    src={
      "https://images.unsplash.com/photo-1550525811-e5869dd03032?ixlib=rb-1.2.1&auto=format&fit=facearea&facepad=2&w=256&h=256&q=80"
    }
  >
    <h4 className="text-lg font-bold">Lorem ipsum</h4>
    <p className="mt-1">
      Lorem ipsum dolor sit amet, consectetur adipiscing elit. Praesent non
      dapibus massa.
    </p>
  </Media>
</div>
```
