# Table

> Managed Table with filter, ordering and paging

<Panel layout="warning" className="p-4 my-5 font-bold">
  Warning: This is an experimental Component, this API may be changed in
  future
</Panel>

```jsx title="Import"
import { Table } from "@webeetle/windy";
```

## Basic

```jsx live
<div className="flex w-full space-x-5 justify-center items-center">
  <div className="w-3/4">
    <Table
      columns={[
        {
          label: "Custom Column",
          value: "actions",
          renderComponent: (row, value) => (
            <div>
              {row.data.name} had {row.data.age} years
            </div>
          ),
          ordered: false,
        },
        { label: "Name", value: "name", ordered: false },
        { label: "Surname", value: "surname", ordered: false },
        { label: "Age", value: "age", ordered: false },
        { label: "Rank", value: "rank", ordered: false },
      ]}
      data={[
        {
          name: "John",

          age: "33",
```

## Filter + Paging + Ordering

```jsx live
<div className="flex w-full space-x-5 justify-center items-center">
  <div className="w-3/4">
    <Table
      columns={[
        {
          label: "Custom Column",
          value: "actions",
          renderComponent: (row, value) => (
            <div>
              {row.data.name} had {row.data.age} years
            </div>
          ),
          ordered: false,
        },
        { label: "Name", value: "name", ordered: false },
        { label: "Surname", value: "surname", ordered: true },
        { label: "Age", value: "age", ordered: true },
        { label: "Rank", value: "rank", ordered: true },
      ]}
      data={[
        {
          name: "John",
          surname: "Cook",
          age: "33",
          rank: "1",
        },
        {
          name: "Mik",
          surname: "Red",
          age: "34",
          rank: "2",
        },
        {
          name: "Nik",
          surname: "White",
          age: "35",
          rank: "3",
        },
        {
          name: "John",
          surname: "Wick",
          age: "36",
          rank: "4",
        },
        {
          name: "Jiik",
          surname: "Book",
          age: "36",
          rank: "4",
        },
      ]}
      color="green"
      paged={true}
      pageSize={3}
      search={true}
    />
  </div>
</div>
```
