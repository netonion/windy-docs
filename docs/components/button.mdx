# Button

> Buttons allow users to take actions, and make choices, with a single tap.

```jsx title="Import"
import { Button } from "@webeetle/windy";
```

## Contained Button

```jsx live
<div className="flex w-full justify-center items-center">
  <Button>Default Button</Button>
</div>
```

## Bordered Button

```jsx live
<div className="flex w-full justify-center items-center">
  <Button layout="bordered">Default Button</Button>
</div>
```

## Text Button

```jsx live
<div className="flex w-full justify-center items-center">
  <Button layout="text">Default Button</Button>
</div>
```

## Custom Button

```jsx live
<div className="flex w-full justify-center items-center">
  <Button
    color="none"
    layout="text"
    className="text-white hover:text-gray-200 text-lg px-4 py-2 bg-gradient-to-l from-red-500 to-yellow-300"
  >
    Default Button
  </Button>
</div>
```
